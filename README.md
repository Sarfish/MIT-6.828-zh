# MIT-6.828-zh

MIT 6.828 操作系统课程的中文翻译

翻译的目标是 [MIT 6.828 Fall 2018](https://pdos.csail.mit.edu/6.828/2018/schedule.html)。

MIT 6.828 是世界上最好的操作系统课程，里面有大量优质的学习材料，所以我希望可以翻译成中文，让更多的人可以学习操作系统。

会先翻译课程实验和课程作业，因为我觉得这是课程的精华。
